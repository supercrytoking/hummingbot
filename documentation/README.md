# hummingbot-docs
Docs for hummingbot, the open source crypto market making bot

## Dependencies

### MkDocs
```
pip install mkdocs
```

### Material for MkDocs
```
pip install mkdocs-material
```

### PyMdown Extensions
```
pip install pymdown-extensions
```

## Develop
```
mkdocs serve
```

Go to `http://localhost:8000`

## Build
```
mkdocs build
```

## Deploy
Drag the `/site` directory to a Netlify repo
